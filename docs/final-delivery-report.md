## 🚀 使用示例

### 基础扫描
```bash
# 扫描单个主机
fg-qimen -H 192.168.1.1

# 扫描C段（默认133端口）
fg-qimen -H 192.168.1.0/24

# 指定端口
fg-qimen -H 192.168.1.1 -p 22,80,443,3306
```

### 端口组扫描（新功能）
```bash
# Web服务扫描（209个端口）
fg-qimen -H 192.168.1.0/24 -p web

# 数据库扫描（18个端口）
fg-qimen -H 192.168.1.0/24 -p db

# 混合格式
fg-qimen -H 192.168.1.0/24 -p web,3306,8000-8010

# 全端口扫描
fg-qimen -H 192.168.1.1 -p all
```

### 代理扫描（新功能）
```bash
# SOCKS5代理
fg-qimen -H 192.168.1.0/24 --socks5 127.0.0.1:1080

# SOCKS5带认证
fg-qimen -H 192.168.1.0/24 --socks5 socks5://user:pass@host:1080

# HTTP代理
fg-qimen -H 192.168.1.0/24 --proxy http://127.0.0.1:8080

# 网卡绑定（VPN场景）
fg-qimen -H 192.168.1.0/24 --iface 10.8.0.2
```

### 项目工作区
```bash
# 创建项目（持久化）
fg-qimen -p my-project -H 192.168.1.0/24

# 增量续扫
fg-qimen -p my-project --resume

# 查看项目列表
fg-qimen projects list
```

### 凭据测试
```bash
# 使用内置字典
fg-qimen -H 192.168.1.0/24 -p 22,3306

# 自定义字典
fg-qimen -H 192.168.1.0/24 -u user1,user2 -P pass1,pass2

# 字典文件
fg-qimen -H 192.168.1.0/24 --user-file users.txt --pass-file passwords.txt
```

### 输出控制
```bash
# 指定输出文件
fg-qimen -H 192.168.1.0/24 -o result.txt -j result.json

# 静默模式（仅输出结果）
fg-qimen -H 192.168.1.0/24 --silent

# 禁用TUI（CI/CD场景）
fg-qimen -H 192.168.1.0/24 --no-tui
```

## 🔒 安全边界

### 严格排除的功能（硬性策略）

**FG-QiMen明确不做**：
- ❌ MS17-010永恒之蓝利用
- ❌ SMBGhost利用
- ❌ Redis写公钥/写shell
- ❌ SSH命令执行
- ❌ 任何CVE漏洞利用
- ❌ 反序列化攻击
- ❌ 后渗透模块

**仅提供**：
- ✅ 端口开放检测
- ✅ 服务识别
- ✅ 版本探测
- ✅ 凭据验证（不执行命令）
- ✅ 未授权访问检测（不利用）
