#### 实现的验证
| 验证函数 | 功能 | 防护 |
|---------|------|------|
| `ValidateHost` | IP/CIDR/范围/主机名 | 防止无效格式崩溃 |
| `ValidatePort` | 端口范围1-65535 | 防止端口溢出 |
| `ValidatePortString` | 端口字符串解析 | 防止非数字输入 |
| `ValidateThreads` | 线程数1-10000 | 防止资源耗尽 |
| `ValidateTimeout` | 超时值验证 | 防止异常超时 |
| `SanitizeFilePath` | 路径安全检查 | 防止目录遍历 |

#### 安全加固效果
```bash
# 防止无效CIDR
$ fg-qimen -H 192.168.1.0/33
Error: invalid CIDR notation (netmask out of range)

# 防止端口溢出
$ fg-qimen -p 99999
Error: port 99999 out of range (1-65535)

# 防止资源耗尽
$ fg-qimen -t 50000
Error: threads too large (max 10000)

# 防止路径遍历
$ fg-qimen -o ../../../etc/passwd
Error: path contains directory traversal
```
